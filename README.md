# Royal River GitHub Pages site

This folder is a standalone one-page Telegram join site for GitHub Pages.

## Files to upload

- `index.html`
- `assets/hero.webp`
- `assets/logo.webp`

## Publish as a GitHub Pages user or organization site

1. Create a public GitHub repository named exactly `OWNER.github.io`.
   - If the GitHub owner is `royalriver`, use `royalriver.github.io`.
   - If the GitHub owner is `royalriverclub-hub`, use `royalriverclub-hub.github.io`.
2. Upload everything inside this `github-pages` folder to the repository root.
3. Open `https://OWNER.github.io` after GitHub Pages finishes publishing.

## Publish from another repository

1. Upload everything inside this `github-pages` folder to the repository root.
2. In GitHub, open `Settings > Pages`.
3. Choose `Deploy from a branch`, then select the default branch and `/root`.
4. Open the Pages URL shown by GitHub.

The Join Telegram button currently points to `https://t.me/royalriver`.
